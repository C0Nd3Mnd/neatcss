About

A minimalist, long lasting, HTML template.

Neat.html is an oppinionated HTML template for writing. It's designed to last as well as to be small.

I like my web pages neat. Neat is a minimalist, long lasting, HTML and CSS template. Inspired by other minimalist sites like Hundred Rabits.

Neat is currently 7k (yes kilobytes) including this documentation. It adds only about 1k of overhead to whatever you write with it.

Getting Started

Copy neat.html to index.html and edit it in your favorite text editor.

Make your own changes to the custom.css file. That way you can update neat.css and keep all your modifications intact.

Design Decisions

As with anything, you've got to make decisions on the design of Neat.html. Here are some of the decisions I've made and the rational for them.

Max Width

The body max-width is 800px by default. I find a thinner column of text to be more readable on very large monitors. This is simple to change by editing the max-width of the body element in neat.css.

Left Aligned

The body is left aligned. I tried center aligned first, but I just wasn't feeling it. I visited some of my favorite sites on the web and found that most of them where left aligned. So, I went with that.

Images

Images are 100% width by default.

Left Gutter

The left gutter is wider at larger sizes, giving the page a little more breathing room.

Buttons are Links

One of the goals is to use symantic web tags. Buttons are a unique case because most people want a button to link somewhere, but the button tag doesn't currently support the href attribute. Buttons, in Neat are anchor tags with a class of button.

<a class="button" href="#">Button</a>

No Header

Originally, I started to build a header with navigation links in it, but the css code piled up and it didn't feel minimalist to me. Those links are also a distraction on whatever message your page is trying to tell. In my research I found lots of examples of minimalist sites without top navigation and only a link back to the home page. As a result, there is no header and no navigation.

Contributing

Neat is distributed as open source under the MIT license.

There are lots of ways you could contribute to Neat. First, please help spread the word about this project. Something as simple as sharing it on social media goes a long way. If you find a problem, please create an issue on Github. I'm happy to review code change requests. To do that, fork the project and create a pull request. Here's a high level overview of how you do that.

  • Fork the project on Github
  • Clone your fork to your local machine
  • Create a new branch
  • Make and test your changes
  • Add and commit those changes
  • Push your new branch up to Github
  • Use the Compare & pull request button on Github to send a PR
